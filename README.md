# Mock Location Pro

A modern Android Mock GPS application built with Java, XML, Material 3, and Google Maps.

Mock Location Pro allows users to set virtual GPS locations, simulate movement, create routes, manage favorite locations, and control location spoofing through a modern Material Design interface.

Features

Interactive Google Maps

- Search locations by name
- Search using latitude and longitude coordinates
- Open locations from Google Maps links
- Draggable location marker
- Long press location selection
- Multiple map modes
  - Normal
  - Satellite
  - Terrain
  - Hybrid

Mock GPS Engine

- Set custom GPS location
- Real-time location spoofing
- GPS and Network provider support
- Background location simulation
- Foreground service support
- Start, Pause, Resume, and Stop controls

Route Simulation

- Walking mode
- Driving mode
- Cycling mode
- Custom speed control
- Route looping
- Route reversal

Floating Joystick

- Overlay joystick control
- Live location movement
- Adjustable speed
- Adjustable transparency
- Invert controls option

Favorites & History

- Save favorite locations

- Fast location reuse
- Search favorites
- Search history
